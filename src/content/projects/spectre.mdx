Spectre는 Astro 기반의 터미널 스타일 테마로, Astro와 TypeScript로 제작되었습니다.

이 블로그의 기반이 되는 테마입니다.

## 주요 기능

- Lighthouse 100 / 100 성능
- 모든 화면 크기에 반응형
- 완전한 접근성 지원
- 타입 안전성
- 자동 사이트맵 생성
- Markdown / MDX 지원
- Content Collections 기반
- [pagefind](https://pagefind.app) 기반 검색
- [giscus](https://giscus.app) 기반 댓글 (비활성화 가능)

## Lighthouse

Spectre는 모든 Lighthouse 항목(성능, 접근성, 권장사항, SEO)에서 100점을 달성합니다:

![Lighthouse 테스트 전 항목 100/100 점수](/img/lighthouse.png)

## 빠른 속도

Spectre는 Tailwind를 포함한 추가 프레임워크를 사용하지 않아 빠른 로딩과 빌드 속도를 제공합니다.

## 설정 가능

Spectre의 기능들은 모두 켜고 끄거나 수정할 수 있습니다. 예를 들어 댓글 기능이 필요 없다면 간단히 비활성화할 수 있습니다.
